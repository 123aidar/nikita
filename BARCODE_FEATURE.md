# Функционал штрих-кодов для товаров

## ✅ Реализованные функции

### 1. Модель продукта
- ✅ Добавлено поле `barcode` (13 символов, уникальное)
- ✅ Автоматическая генерация штрих-кодов EAN-13 для всех товаров
- ✅ Штрих-коды начинаются с 200-299 (внутренние коды магазина)

### 2. API для работы со штрих-кодами
- ✅ **GET** `/sales/api/search-barcode/?barcode=<код>` - поиск товара по штрих-коду
- ✅ Возвращает информацию о товаре (ID, название, цена с наценкой, остаток)
- ✅ Проверяет наличие товара на складе

### 3. Форма создания продажи с поддержкой сканера
- ✅ Поле для сканирования штрих-кодов в верхней части формы
- ✅ Автоматическое добавление товара при сканировании
- ✅ Поддержка ручного ввода штрих-кода
- ✅ Автоматический поиск при вводе 8+ символов (для быстрых сканеров)
- ✅ Поиск по клавише Enter
- ✅ Увеличение количества, если товар уже добавлен
- ✅ Визуальная подсветка добавленных/обновленных товаров
- ✅ Уведомления об успешном добавлении
- ✅ Автофокус на поле штрих-кода для быстрой работы

### 4. Административная панель
- ✅ Отображение штрих-кода в списке товаров
- ✅ Поиск по штрих-коду в админке
- ✅ Поле штрих-кода в форме редактирования товара

### 5. Детальная страница товара
- ✅ Отображение штрих-кода
- ✅ Кнопка "Печать штрих-кода" для печати этикеток
- ✅ Генерация изображения штрих-кода в формате PNG

### 6. Печать этикеток
- ✅ **GET** `/products/<id>/barcode-label/` - страница печати этикетки
- ✅ **GET** `/products/<id>/barcode-image/` - PNG изображение штрих-кода
- ✅ Красивый шаблон этикетки с информацией о товаре
- ✅ Оптимизация для печати

### 7. Management команды
- ✅ `python manage.py generate_barcodes` - генерация штрих-кодов для товаров без них
- ✅ `python manage.py generate_barcodes --regenerate` - перегенерация для всех товаров

## 📦 Установленные зависимости

```
python-barcode[images]  # Генерация штрих-кодов с поддержкой изображений
```

## 🚀 Как использовать

### 1. Создание новой продажи со сканером

1. Перейдите в **Продажи → Новая продажа**
2. В верхней части формы будет поле **"Сканер штрих-кодов"**
3. Наведите сканер на штрих-код товара или введите код вручную
4. Товар автоматически добавится в список (или увеличится количество)
5. Продолжайте сканировать другие товары
6. Нажмите **"Зарегистрировать продажу"**

### 2. Печать этикетки со штрих-кодом

1. Откройте страницу товара
2. Нажмите кнопку **"Печать штрих-кода"** (фиолетовая кнопка)
3. Откроется страница с этикеткой
4. Нажмите **"🖨️ Печать"** или Ctrl+P

### 3. Просмотр штрих-кода в админке

1. Войдите в админ-панель Django
2. Откройте список товаров
3. Штрих-код отображается в столбце
4. Можно искать товары по штрих-коду

## 🔧 Технические детали

### Формат штрих-кода
- **Стандарт**: EAN-13 (13 цифр)
- **Префикс**: 200-299 (внутренние коды)
- **Контрольная сумма**: автоматический расчет по стандарту EAN-13

### API Endpoint детали

**Поиск по штрих-коду:**
```
GET /sales/api/search-barcode/?barcode=2001234567890
```

**Ответ (успех):**
```json
{
    "success": true,
    "product": {
        "id": 1,
        "name": "Молоко 3.2%",
        "barcode": "2001234567890",
        "price": 89.70,  // с наценкой 15% и НДС 20%
        "quantity": 50.0,
        "unit": "л"
    }
}
```

**Ответ (ошибка):**
```json
{
    "success": false,
    "error": "Товар со штрих-кодом \"2001234567890\" не найден"
}
```

### Работе со сканерами

Система поддерживает работу с USB и Bluetooth сканерами штрих-кодов:
- Сканер работает как клавиатура (вводит цифры)
- Автоматическое распознавание при вводе 8+ символов
- Задержка 100мс для завершения сканирования
- Можно использовать Enter для подтверждения

## 📝 Миграции

- Автофокус для быстрого начала работы

## ⚠️ Важные примечания

1. **Уникальность**: Каждый товар имеет уникальный штрих-код
2. **Генерация**: При создании нового товара штрих-код не генерируется автоматически - нужно либо ввести вручную, либо запустить команду `generate_barcodes`
3. **Формат**: Штрих-код должен быть ровно 13 цифр (EAN-13)
4. **Сканеры**: Работает с любыми USB/Bluetooth сканерами, эмулирующими клавиатуру
5. **Цены**: API возвращает цену с учетом наценки магазина (15%) и НДС (20%)
