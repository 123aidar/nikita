## 🔍 ПРОВЕРКА РАБОТЫ ЧАТА В БРАУЗЕРЕ

1. Откройте браузер: http://127.0.0.1:8000/
2. Войдите под пользователем (логин: admin, пароль: admin)
3. В правом нижнем углу должна быть зеленая кнопка чата
4. Нажмите на кнопку - откроется виджет чата
5. Напишите сообщение и отправьте
6. Откройте другую вкладку браузера с тем же сайтом
7. Войдите под другим пользователем
8. Откройте чат - вы должны увидеть историю сообщений
9. Отправьте сообщение - оно должно появиться в первой вкладке в реальном времени

## 🐛 ОТЛАДКА

### Если чат не работает:

1. **Проверьте консоль браузера (F12)**
   - Не должно быть ошибок JavaScript
   - Должно быть сообщение о подключении WebSocket

2. **Проверьте консоль сервера**
   - Не должно быть ошибок импорта
   - При отправке сообщения должны быть логи

3. **Проверьте WebSocket подключение**
   - В консоли браузера (F12 → Network → WS)
   - Должно быть подключение к ws://127.0.0.1:8000/ws/chat/
   - Статус: 101 Switching Protocols

### Если видите кракозябры вместо русского текста:

1. Убедитесь, что все Python файлы начинаются с `# -*- coding: utf-8 -*-`
2. Запустите `python check_encoding.py`
3. Убедитесь, что в браузере установлена кодировка UTF-8
4. Проверьте консоль: запустите `chcp 65001` перед запуском сервера

## ⚠️ ВАЖНО

1. **НЕ используйте** `python manage.py runserver --noreload` - это отключит WebSocket
2. **Используйте** только `python manage.py runserver` или `start_server.bat`
3. **WebSocket** работает только при запущенном сервере
4. **In-memory channel layer** не сохраняет сообщения между перезапусками сервера
